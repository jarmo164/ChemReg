# Source assignment

## Canonical sources of truth
- Repo code: implementation truth
- `README.md`: high-level product + architecture truth
- `wiki/*.md`: working documentation truth for MVP boundary, auth status, testing, deployment
- `openspec/changes/close-mvp-readiness-gaps/*`: change scope and verification checklist

If older documents disagree with the code, the repo stack wins unless the team explicitly chooses a replacement architecture.
